# EDY Istanbul 2026

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

- Hero section is complete with scroll-based parallax via `useHeroParallax` and cloud animations via `useGSAP`.
- Tailwind styles are managed centrally rather than per-breakpoint.
- Assets are placeholders — no optimization done yet.
- Feel free to experiment with the CSS; the current laptop layout is the preferred baseline.
